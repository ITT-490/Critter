# Critter - IoT Device Network Scanner

Critter is a Python-based GUI network scanner designed to discover active IoT devices on your local network.

## Features
- Scan a range of IP addresses
- Multi-port check (default: 22 and 445)
- Reverse DNS lookup to display hostnames
- Threaded scanning for faster results
- GUI with color-coded output

## Requirements
- Python 3.x
- Tkinter (usually comes with Python)

## Usage
1. Run `critterV2.3.py`
2. Enter start and end IP addresses
3. Click "Start Scan" to see live results
4. Active hosts will appear in green with hostname
